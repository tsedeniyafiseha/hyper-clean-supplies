# Product Overview

Hyper Cleaning Supplies is a fully responsive e-commerce platform specializing in professional cleaning materials and supplies. The application provides a complete online shopping experience with product browsing, cart management, user authentication, order processing, and admin capabilities.

## Core Features

- Product catalog with categories (cleaning chemicals, bathroom care, kitchen care, floor care, dispensers, gloves, paper products)
- Shopping cart with Redux state management
- User authentication (credentials, Google, GitHub OAuth)
- Email verification system
- Product reviews and ratings
- Wishlist functionality
- Stripe payment integration
- Admin dashboard for managing products, categories, and orders
- Responsive design optimized for mobile and desktop

## Business Domain

The platform focuses on B2B and B2C sales of professional cleaning supplies, with emphasis on product discovery, detailed product information, and streamlined checkout experience.
